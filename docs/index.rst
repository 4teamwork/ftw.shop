Welcome to ftw.shop's documentation!
====================================

ftw.shop aims to become a general purpose webshop product for Plone.
It features item variations, an extensible checkout wizard, pluggable
payment processors and optional SQLAlchemy storage.
 
Currently supported versions of Plone are Plone 3 and Plone 4.
